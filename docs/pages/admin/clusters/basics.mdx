## Remove Clusters
```bash
devspace remove cluster [cluster-name]
```

:::warning Data Loss
**Removing a cluster is irreversible.** You will <u>**not**</u> be able to restore information about Spaces, cluster users, limits etc. even if you reconnect the cluster again.
:::
